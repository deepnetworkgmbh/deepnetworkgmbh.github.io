Full-Stack Dev m/f/d

You’d bolster our backbone in our cloud migration projects, using these technologies amongst others:

  • Kubernetes and Docker
  • mostly RDBMS (with T-SQL or U-SQL)
  • Azure Cloud Stack
  • C# and Java/Spring

Our company has been founded 5 years ago and runs as remote-only. Meetings are scheduled as needed and we communicate efficiently via Teams Video Calls, Slack and E-Mail. Workdays don’t follow a structure; you’ll have a lot of freedom but are responsible for adjusting it to customer and meeting requirements. You’ll be goaled realistically on customer happiness and project success.

Requirements

  1. Relevant industry experience building and running (micro-)services (Azure preferred)
  2. C#, C++ or Java experience to read, modify and change existing code bases
  3. Good understanding of RDBMS and/or NoSQL databases
  4. Bonus experience: Azure Cloud Stack (EventHub, Service Bus, App Insights), Azure DevOps, Python, ASP.NET/MVC, TypeScript, CSS, jQuery

Benefits :

  • A friendly, helpful team
  • Flexible work times
  • Startup culture
  • Full remote
